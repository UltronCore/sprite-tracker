PART 1

What it does

Five things, all inside Discord. Members keep using the server normally — the bot handles trust, trades, and collections in the background.

🤝 Trust & reputation

After a good trade, members vouch for each other. Vouches build trust and automatically promote traders up flair ranks. Earns the verified-trader role automatically.

/vouch+rep/profile/rank/leaderboard

🔁 Safe two-party trades

A trade only counts when both people press Confirm — so nobody gets scammed on the drop. Then it prompts both to vouch. Everything is logged.

/trade

📒 Collection tracking

Members track all 41 sprites on a companion web page, then sync it into Discord with one code. The bot makes shareable collection and "looking for" images automatically.

/synccollection/mycollection/missing/spriteset

🔎 Trade matchmaking

The bot knows who has what, so it can instantly find trade partners and show server-wide progress and the most-wanted sprites.

/holders/spritematch/spriteinfo/guildprogress

📋 Hand-off queues

For rare sprites (Zero Point, John Wick…): a distributor gives them out one at a time in a fair line. Members click Join on a live board — the bot tracks positions and pings whoever's next. Open only the queues you want.

/queue open/queue next/queue done

🛡️ Safety & moderation

Report scammers to a private mod channel, blacklist bad actors (it strips their trust roles), and stop alt-account farming with account-age, server-tenure, and cooldown checks.

/reportscammer/blacklist

🎉 Automatic housekeeping

Welcomes new members, keeps the sprite-holder lists current, posts a daily leaderboard, an optional weekly digest, and announces brand-new sprites when Epic adds them.

/digest/whohas/whoneedswelcome
🖱️ One-click control panel: a pinned message of buttons (My Collection, Find Trades, Join a Queue, Sync…) means members rarely have to type a command at all — they just click. Set it up once with /panel.
The big idea: the bot turns a normal trading server into a tracked, trusted, scam-resistant community — members see their collection, find partners fast, and build a reputation that's hard to fake.
PART 2

How to add it — step by step

About 30 minutes, no coding. You'll create the bot, invite it, host it, and run one setup command. Anyone you trust with a computer can follow this.

1

Create the bot application

Go to the Discord Developer Portal → New Application → name it "Sprite Trade Stop". Then open Bot in the left menu → Add Bot.

2

Turn on two switches (intents)

Still on the Bot page, scroll to Privileged Gateway Intents and turn ON:

  • Server Members Intent — lets it welcome people and manage roles
  • Message Content Intent — lets the +rep shortcut and analytics work

Click Save Changes.

3

Copy the secret token

On the Bot page click Reset Token → Copy. This is the bot's password — keep it private, you'll paste it once during hosting (Step 5).

⚠️ Never share the token publicly. If it ever leaks, click Reset Token to make a new one.
4

Invite it to the server

Open OAuth2 → URL Generator. Tick bot and applications.commands. Then under permissions tick: Manage Roles, Send Messages, Embed Links, Attach Files, Read Message History, Add Reactions. Copy the link at the bottom, open it, and add the bot to Sprite Trade Stop.

Important — role position: in Server Settings → Roles, drag the bot's role above all the sprite, Gold, flair, and verified-trader roles. Discord only lets a bot manage roles that sit below its own.
5

Host it (so it's online 24/7)

The bot is a small program that needs to stay running. Pick one — the full click-by-click guide is in HOSTING.md:

Oracle Cloud — Free

$0 forever

Best free option. A real always-on server. Most setup, fully free.

Bot-Hosting.net — Free

$0 (claim daily)

Easiest. Upload the folder, paste the token in a web panel. No Linux.

Hetzner / DigitalOcean

~$4–5 / month

Most reliable, set-and-forget. A small paid server.

During hosting you'll paste the token from Step 3 into a file called .env (or a token box in the panel).

6

Run one setup command

In Discord, type /setup. The bot automatically finds your channels (trade-portal, vouch-trades, modlog, leaderboard, welcome, news, …) and your sprite/Gold/flair/verified roles by name, and saves them. It tells you if anything's missing so you can create it and run /setup again.

No hunting for IDs by hand — /setup does the wiring. Optionally turn on the weekly recap with /digest on.
✅ That's it. The bot is live. Members can /synccollection, /trade, and /vouch right away.

PART 5

Why it's safe to run

💸 No cost, no AI bills

Python + a small local file. No paid APIs, no subscriptions. Free hosting works fine.

🔒 Your server only

Locked to your one server. The bot can't be pulled into someone else's server to leak your data.

🚫 Anti-scam & anti-spam

Two-party confirms, account-age + tenure checks, cooldowns, vouch caps, and a blacklist that removes trust roles.

📖 Open & documented

Full source on GitHub, automated tests, and a complete handoff guide so any helper can maintain it.
